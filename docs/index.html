Introduction

Description

PickleRick is a multi-purpose Discord bot with lots of features from moderation to having fun with Google Images.
It is a go-to Discord bot that you can use for your own Discord server.
Its main purpose is for Admins and Moderators to moderate their servers and manage 100s of members easily.
With easy-to-type commands and explanative help embeds, moderation has never been easier.

Not an Admin or Mod?
Fret not, I have fun commands as well like betting on a flip coin, answering trivia questions,
getting statuses on games like Minecraft and Fortnite, doing math and much much more!

Moderation

Commands Like pr!mute, pr!poll, pr!inv and pr!kick allow mods to do a whole range of stuff in the server.
You can also change some server and bot settings using the pr!ss command, a very advanced command for you to change important settings easily.

Some commands require you to have special permissions, like the pr!inv command (used for creating an invite), which requires to have the Create Instant Invite permissions.
